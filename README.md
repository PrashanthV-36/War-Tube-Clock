# War Tube Clock

The **War Tube Clock** is a creative digital clock project built using **HTML, CSS, and JavaScript**, designed with a unique war-inspired tube effect. Instead of showing time in a traditional way, this project gives a **futuristic battlefield vibe**, where digits appear inside glowing tube-like structures with an animated war theme.

## Features

* ⏰ Real-time digital clock display (hours, minutes, seconds).
* ⚡ Tube-like glowing design for each digit.
* 🎨 War-inspired styling with bold colors and effects.
* 🌐 Built using only HTML, CSS, and JavaScript (no external frameworks).

## Use Cases

* Perfect for **web design practice**.
* Can be used as a **desktop clock widget**.
* A fun addition to **GitHub projects** showcasing creative time displays.

## How It Works

1. JavaScript fetches the **current system time**.
2. Each digit is displayed in a **styled tube container**.
3. CSS adds the glowing and animated **war-like effects** to enhance the look.
